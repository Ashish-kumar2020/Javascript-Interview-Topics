# 27 - Debouncing

Debouncing is a technique used to limit the rate at which a function gets executed.

The function will execute only after a specified delay once the event stops triggering.

Example Use Cases

- Search Bar
- Auto Suggestion
- Resize Event
- Input Validation

If a user types continuously, the function execution keeps getting delayed.

The function executes only after the user stops typing for the specified delay.

Debouncing improves performance by reducing unnecessary function calls.
