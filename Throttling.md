# 28 - Throttling

Throttling is a technique used to control how many times a function can be executed within a given time interval.

Unlike Debouncing, throttling ensures that the function executes at regular intervals.

Example Use Cases

- Scroll Events
- Mouse Move Events
- Window Resize Events
- Button Clicks

Even if an event triggers multiple times, throttling limits execution frequency.

Throttling improves performance and prevents excessive function execution.
