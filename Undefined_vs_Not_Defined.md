# 6 - Undefined vs Not-defined

Both undefined and not defined are different things, they are not same

undefined - it is used during the memory component phase when a variable gets initialised in the memory and didn't gets assigned with any value, at that time js assigns a special keyword or placeholder named undefined

not defined - it is used when javascript did not find the variable inside the scope or in the parent scope, in this case js throws an error stats that the variable or function is not defined

NOTE : JavaScript is a loosely typed language, it means it does not attaches its variable with specific data type
